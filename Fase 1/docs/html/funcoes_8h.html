Retorna
Lista ligada de antenas ou NULL se erro

Carrega antenas de um ficheiro de texto.

Parâmetros
nome_ficheiroCaminho para o ficheiro contendo os dados das antenas
Retorna
Lista encadeada de antenas ou NULL em caso de erro

O ficheiro deve seguir o formato:

  • Primeira linha: total_linhas total_colunas
  • Linhas subsequentes: representação do grid ('.' para vazio)
    Aviso
